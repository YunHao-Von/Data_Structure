1.2.1assert

assert语句成为断言，判断asseert后面紧跟的语句是True/False，如果是True则继续向下执行语句，如果是False则中断程序，抛出AssertionError的异常，同时输出assert False语句逗号后面的提示信息。 

def Sum(n):
    assert n>=1,"参数n应该大于等于1"
    return n*(n+1)//2  # 双除号表示整除
n = -5
Sum(n)

1.2.2.3enumerate

enumerate()函数用于将一个可以遍历的数据对象组合成为一个索引序列，同时列出数据和数据的下标，一般用子啊for循环当中。 

a = ["data1","data2","data3"]
for data in enumerate(a):
    print(data)

for data in enumerate(a,5):
    print(data)

返回: 

(0, 'data1')
(1, 'data2')
(2, 'data3')
(5, 'data1')
(6, 'data2')
(7, 'data3')

1.2.2.4列表推导式

1.2.2.5对列表元素做映射

map()会根据提供的函数对指定列表做映射。其语法格式如下: 

map(function,iterable,...)

function指定一个函数，iterable指定了一个或者多个列表，其返回值是一个迭代器，可以通过list()来转化为列表。 

def square(x):
    return x**2
a = [1,2,3,4,5]
b = map(square,a)
print(list(b))
v1 = [1,3,5,7,9]
v2 = [2,4,6,8,10]
v3 = map(lambda x,y:x+y,v1,v2)
print(list(v3))

返回: 

[1, 4, 9, 16, 25]
[3, 7, 11, 15, 19]

1.3深浅拷贝

1.3.1非复制方法————直接赋值

a = [0,1,2]
print(a)
b=a
print(b)
a[0] = 4
print(a)
print(b)

打印输出结果为： 

[0, 1, 2]
[0, 1, 2]
[4, 1, 2]
[4, 1, 2]

在执行这种操作后，a和b相当于C++语言中的指针，它们指向相同的存储空间，此后同步改变，这种复制并没有实现列表的真复制。

1.3.2列表的深复制

import copy
a = [1,[1,2,3],4]
b= copy.deepcopy(a)
print(a)
print(b)
a[0] = 99
b[0] = 101
print(a)
print(b)

1.3.3列表的浅复制

import copy
a = [1,[1,2,3],4]
b= a.copy()
print(a)
print(b)
a[1][0] = 99
print(a)
print(b)

这种情况下，对于a的第一层实现了深复制，但是对于嵌套的列表而言仍然是浅复制，因为内层的列表实际上保存的是地址，在复制a的时候把地址也复制过去了。 实际上如果使用列表推导式生成新的列表也是一种浅复制方法，只对第一层实现了深复制。

1.4Python的面向对象

1.4.1类变量和类方法

python对象的变量分为类变量和实例变量两种，类变量在所有实例中共享，定义在类中而且在所有方法之外，一般类变量通过类对象来引用。
实例变量则是对每个实例所独有的数据，一般实例变量通过实例对象引用。
属性（包含变量和方法）又分为共有属性和私有属性，私有属性名称以两个下划线开头，私有属性在类的外部无法直接访问。 

class A:
    x=2  # 类属性x
    __y = 0  # 私有类属性  
    def __init__(self,p): # 构造方法
        self.z = p  # 实例属性z
        self.__w = 0 # 私有实例属性w
    def add(self):
        self.__y += 10
        self.__w += 20
    def disp(self):
        print(A.x, self.__y,self.z,self.__w)

A.x += 1
a = A(10)
a.add()
a.disp()
b = A("Bye")
b.disp()

返回： 

3 10 10 20
3 0 Bye 0

1.4.2方法参数传递

参数为不可变数据：
类型一般为数值、字符串、元组。
在这种情况下参数传递过程采用了值复制的方式，将实参值直接复制给了对应的形参，然后执行被调用的方法，返回时不会改变实参的值。
参数为可变数据：
参数类型为列表、字典和集合。
当参数为可变数据类型的时候，形参的执行结果会传递给实参。
原理：形参的地址不变，结果会回传给实参对象。

1.4.3继承

单继承

子类可以继承父类的所有非私有属性和非私有方法，但是不能继承父类的私有属性和私有方法，所以无法在子类中访问父类的私有属性和调用父类的私有方法。
可以通过在子类中调用父类的公有方法方位父类的私有属性和私有方法。

多继承

class 子类名(父类1，父类2，...，父类n):

需要注意圆括号中的父类的顺序，若父类中有相同的方法名，而在子类使用时没有指定，python会从左到右搜索，查找父类中是否包含该方法。
